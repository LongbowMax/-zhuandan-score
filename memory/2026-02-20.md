## 配置文件

已配置内容：
- Tushare token（已填入）
- 南网储能监控（持仓70%）
- 宏观经济监控启用
- 微信通知配置（需填入SendKey）

## 微信通知配置步骤

1. 访问 https://sct.ftqq.com/
2. GitHub登录获取 SendKey
3. 填入 `config/settings.json`：
   ```json
   "wechat": {
     "enabled": true,
     "send_key": "SCT你的SendKey"
   }
   ```
