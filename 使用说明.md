## 转蛋计分助手 - 使用说明

### 安装方法

1. 下载 APK 文件
2. 在 Android 手机上安装
3. 打开应用

### 基本操作

#### 第一步：添加玩家

1. 在首页点击输入框
2. 输入玩家姓名
3. 点击"+"按钮添加
4. 重复以上步骤添加至少4个玩家

#### 第二步：开始游戏

1. 点击右下角的"开始游戏"按钮
2. 进入计分页面

#### 第三步：设置排名

为每个玩家选择排名：
- 头游（第一名）
- 二游（第二名）
- 三游（第三名）
- 末游（第四名）

#### 第四步：设置家族

- 每个玩家选择一个家族（家族1或家族2）
- 同一家族的2人为一队
- 也可以点击"自动分配"随机分配

#### 第五步：设置特殊牌型

如果有特殊牌型，设置：
- 6张以上炸的数量
- 是否有天王炸

#### 第六步：计算得分

点击"计算并保存"按钮，系统会自动：
- 根据排名判断胜负
- 计算基础分（1/2/3分）
- 计算倍数（炸×2，天王炸×2）
- 更新每个玩家的总分

#### 第七步：查看结果

在历史记录页面可以查看：
- 当前总分排名
- 每局详细得分
- 删除错误的记录

### 计分规则详解

**基础分：**
- 头游+二游同一家 = 赢3分
- 头游+三游同一家 = 赢2分
- 头游+末游同一家 = 赢1分

**倍数计算：**
- 1炸 = ×2
- 2炸 = ×4
- 3炸 = ×8
- 天王炸额外 ×2

**示例：**
- 头游+二游 + 1炸 + 天王炸 = 3 × 2 × 2 = 12分
- 头游+三游 + 2炸 = 2 × 4 = 8分

### 注意事项

1. 每个排名只能有一个玩家
2. 每个家族必须有2人
3. 删除游戏记录会回退分数
4. 数据保存在本地，卸载应用会丢失

### 常见问题

**Q: 可以添加超过4个玩家吗？**
A: 可以，但每局游戏需要恰好4人参与。

**Q: 一个人可以是一家人吗？**
A: 可以，如果你的翻牌结果是同一个人拿到两张同样的牌，他自成一家。

**Q: 如何修改已保存的记录？**
A: 删除错误的记录，然后重新添加。
