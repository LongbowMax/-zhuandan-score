# 转蛋计分助手

一个专为"转蛋"游戏设计的Android计分应用。

## 游戏规则

- 永远打2
- 发牌前翻开一张牌，持有同样两张牌的选手为一家
- 如果一个人拿了两张同样的花色，座位不调整

### 计分规则

| 结果 | 得分 |
|------|------|
| 头游 + 二游（同一家） | 赢3分 |
| 头游 + 三游（同一家） | 赢2分 |
| 头游 + 末游（同一家） | 赢1分 |

### 特殊牌型倍数

- 6张以上炸：每有一炸，分值翻一倍
- 天王炸：分值再翻一倍

## 功能特点

- ✅ 动态添加/删除玩家
- ✅ 自动计算得分
- ✅ 炸和天王炸倍数计算
- ✅ 实时排名显示
- ✅ 历史记录保存
- ✅ 数据本地存储

## 技术栈

- Kotlin
- Jetpack Compose
- Material Design 3
- DataStore（本地存储）

## 构建方法

1. 使用 Android Studio 打开项目
2. 同步 Gradle
3. 运行 `app` 模块

## 使用说明

1. **添加玩家**：在首页添加至少4个玩家
2. **开始游戏**：点击开始游戏按钮
3. **设置排名**：为每个玩家选择头游/二游/三游/末游
4. **设置家族**：选择玩家所属家族（同一家的人组成一队）
5. **特殊牌型**：设置炸的数量和是否有天王炸
6. **计算得分**：系统自动计算并保存
7. **查看历史**：在历史页面查看总分排名和每局详情
