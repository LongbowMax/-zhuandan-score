### 🚀 使用方法

#### 运行监控
```bash
cd stock-risk-monitor
python main_v3.py
```

#### 自动发送规则
- **风险预警邮件**：当股票出现高风险/中风险且你有持仓时自动发送
- **每日监控报告**：每次运行后自动发送日报

#### 邮件发送示例
```python
from stock_risk_monitor.modules.email_notifier import EmailNotifier

notifier = EmailNotifier()

# 发送风险预警
notifier.send_risk_alert(
    stock_name="南网储能",
    stock_code="600995", 
    risk_summary={...},
    all_risks=[...]
)

# 发送日报
notifier.send_daily_report(stocks_data)
```
